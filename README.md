# 📂 Organizador de Arquivos em Python

Projeto de automação desenvolvido em Python para organizar arquivos automaticamente em pastas com base em suas extensões.

---

## 🚀 Funcionalidades

* 📁 Organiza arquivos automaticamente
* 📄 Identifica tipos de arquivos por extensão
* 🧠 Cria pastas automaticamente (se não existirem)
* 📦 Move arquivos para suas respectivas categorias:

  * Imagens
  * Documentos
  * Planilhas
  * Músicas
  * Vídeos
